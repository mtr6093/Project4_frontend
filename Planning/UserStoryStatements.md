# User Story Statments to be translated into functioning code

## MVP Requirements

- (backend)Database for equipment with fields for the following
    * Column for user ID
    * Column for type (i.e. tractor, combine, sprayer, etc)
    * Column for Model (Equipment model number i.e. 4020, 8410R, etc.)
    * Column for hours as of last engine oil/filter change
    * Column for hours as of last hydaraulic filter change
    * Column for hours as of last fuel filter change
    

- As a user I want to see a list of all equipment 
- As a user I want to be able to see the service details for each model by selecting that model on screen
- As a user I want to be able to update the service hours for each field


## Silver Requirements

- As a user I want to be able to enter new models into the equipment list
- As a user I want to be able to delete models from the list

## Gold Requirements

- As a user I want to be able to have an individual account (requires frontend and backend changes)
- As a user I only want to see my equipment, not anyone elses (requires front end filtering at a minimum)
- As a user I also want to track service hours for the following: Engine Air Filter, Cabin Air Filter, Front Differential oil, Final Drive oil, Reverser oil (requires frontend and backend changes)
